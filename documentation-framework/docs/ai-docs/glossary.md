This document defines the key terms, acronyms, and product concepts used across this project's documentation and codebase.

## Template for Adding Terms

When adding new terms to this glossary, use this template:

```markdown
### Term Name
- **Definition**: Concise, plain-English explanation of the term.
- **Scope / Target Audience**: Who uses or encounters this term? (e.g., Developers, Admins, End Users).
- **Related Modules**: Links to source code directories or relevant documentation folders.
- **Notes / Examples**: Any extra context, common edge cases, or configuration details.
```
